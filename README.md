## Install
```
npm i plain-app -S
```

## Quick Start
```
import Vue from 'vue'
import PlainApp from 'plain-app/dist/plain-app.umd.min'
import 'plain-app/dist/plain-app.css'

Vue.use(PlainApp)
```
